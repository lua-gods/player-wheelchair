Replaces the player controller with a wheelchair. Absolutely horrible to control.

> [!NOTE]
> This avatar requires the SillyPlugin  
> [you can get it at Modrinth](https://modrinth.com/mod/figura-sillyplugin)

## Controls
all walking keys are removed and replaced with the following:

| Key | Action |
| --- | --- |
| `A` | Move Left Wheel |
| `D` | Move Right Wheel |
